What's in the box

Stuff we set up so you don't have to fight the defaults on day one.

Window chrome

Close/minimize/maximize on the left, clean title bars, rounded corners where it makes sense.

Debian underneath

Install software with apt from Debian repos. Familiar, stable base—no rolling-release surprises.

No phone-home stuff

We don't ship analytics or adware. What's on the ISO is what's documented.

Horizon theme

Icons, GTK/Qt tweaks, wallpaper, and sounds aimed at one consistent desktop look.

Dev tools included

Git, build essentials, and common runtimes—enough to clone a repo and get going.

Open source

Configs and scripts are on GitHub. Change them, fork them, ship your own variant.

Install steps

  1. Download the ISO and check the SHA256 hash.
  2. Write it to a USB stick (Etcher, Rufus, or dd).
  3. Boot from USB and follow the installer.
